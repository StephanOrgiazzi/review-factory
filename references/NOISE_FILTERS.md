# Noise filters

Usually ignore:

- lockfiles: bun.lock, package-lock.json, pnpm-lock.yaml, yarn.lock, Cargo.lock, go.sum, poetry.lock, Pipfile.lock, flake.lock
- generated files marked with @generated, codegen, autogenerated, eslint-disable generated blocks
- minified/bundled/source map assets: *.min.js, *.min.css, *.bundle.js, *.map
- snapshots unless the changed snapshot indicates a behavior change worth reviewing
- formatting-only diffs

Do not ignore:

- migrations, even if generated
- generated API types when they reveal contract changes
- lockfiles if dependency versions changed in a security-sensitive way
